Playing with Object & Array Data Selection

mpc
MPC equals self.whatever in the ...controller.js file associated with mpc.
QUESTION: self.whatever always equals an object
In this case, MPC equals self.arrayList. (also .message & .objectApp which are commented out)
self.arrayList is an object holding another object, called arrayList, which is holding an array, called list, which holds object(s) containing all table data as object properties with values.
Therefore mpc displays all table data + name defined afer self. in controller.js file (arrayList) + name defined in factory.js file (list) + brackets & curlys + any other data defined in the controller.js file placed after self. (like .message & .objectApp which are commented out):
{{mpc}}


mpc.arrayList.list
Displays complete array, including brackets [ ], and all table data stored in objects, each within curly braces { }:
{{mpc.arrayList.list}}


mpc.arrayList.list[0].other_notes
Displays the array-zero-index, value from 'other_notes' table column:
Note: The array index order is adjusted with the SQL query which is coded in the routes.js file
{{mpc.arrayList.list[0].other_notes}}


Start of AngularJS repeating data:

Targeting a specific table cell/object property is not normally done with AngJS.
However to target one property within an object use this format:
"(ANY-table-column-name)
in
(controllerAs-value-given).
(controllerJS-file-value-given-after-self-dot).
(FactoryJS-file-value-given-within-the-object-to-collect-the-array-giving-table-data)"
Note: enter data without ()
Example: serial_number in mpc.arrayList.list
